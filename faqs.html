Everything you need to know about using Integral Explorer

Getting started is simple:

  1. Enter a function in the input field (e.g., x^2, sin(x), x*e^x)
  2. Click "Analyze Function" to get technique recommendations
  3. Use hints if you need guidance on the integration process
  4. Enter your answer and click "Check Answer" for instant feedback
  5. View the graph on the right side to visualize the function

No installation required - just open the page and start learning!

Our platform supports five major integration techniques:

Power Rule (Easy)

Formula: $\int x^n dx = \frac{x^{n+1}}{n+1} + C$

Examples: x^2, 3x^4, x^(-2)

Trigonometric (Medium)

Functions: sin, cos, tan, sec, etc.

Examples: sin(x), cos(2x), tan(x)

U-Substitution (Medium)

Pattern: Functions with $f(g(x)) \cdot g'(x)$

Examples: 2x/(x^2+1), x*e^(x^2)

Integration by Parts (Hard)

Formula: $\int u \, dv = uv - \int v \, du$

Examples: x*e^x, x*sin(x), ln(x)

Partial Fractions (Hard)

Method: Decomposition of rational functions $\frac{P(x)}{Q(x)}$

Examples: (x+1)/(x^2-1), 1/(x^2+x-2)

Our smart hint system provides progressive guidance without giving away the answer:

Example: Power Rule Hints
  1. Identify the Pattern: "Look at each term in your polynomial. What power of x does each term have?"
  2. Power Rule Formula: "Remember: ∫x^n dx = x^(n+1)/(n+1) + C, where n ≠ -1"
  3. Don't Forget: "Always add the constant of integration (+C) at the end!"
